Fungsi Kuadrat

Belajar dan pahami tentang fungsi kuadrat, grafik, dan kalkulasinya

Apa itu Fungsi Kuadrat?

Fungsi kuadrat adalah fungsi polinomial yang memiliki variabel dengan pangkat tertinggi dua. Bentuk umum dari fungsi kuadrat menyerupai bentuk persamaan kuadrat.

Persamaan Kuadrat: \( ax^2 + bx + c = 0 \)

Bentuk Umum Fungsi Kuadrat: \( f(x) = ax^2 + bx + c \)

Fungsi Kuadrat: \( f(x) = \) fungsi kuadrat

x: variabel

a, b: koefisien

c: konstanta

a ≠ 0: Koefisien a tidak boleh sama dengan nol.

Rumus Fungsi Kuadrat

Grafik fungsi kuadrat dapat digambarkan dalam diagram Cartesius. Berikut adalah langkah-langkah dalam menggambar grafik fungsi kuadrat berdasarkan persamaan dan tabel nilai:

  • Titik potong terhadap sumbu Y: Diperoleh dengan mengganti x = 0.
  • Titik potong terhadap sumbu X: Diperoleh dengan mencari nilai x yang membuat persamaan sama dengan nol (misalnya menggunakan faktorisasi).

Contoh Persamaan dan Grafik

Misalkan kita punya persamaan kuadrat \( y = x^2 + 2x + 1 \), maka:

  • Titik potong terhadap sumbu Y: \( y = 1 \) ketika \( x = 0 \). Jadi, titik potongnya adalah (0, 1).
  • Titik potong terhadap sumbu X: Solusi dari \( x^2 + 2x + 1 = 0 \) menghasilkan \( x = -1 \). Jadi, titik potongnya adalah (-1, 0).

Hubungan Koefisien dengan Grafik

Grafik fungsi kuadrat juga dipengaruhi oleh koefisien \( a \), \( b \), dan \( c \). Berikut penjelasannya:

Koefisien a:

  • Jika \( a > 0 \), grafik terbuka ke atas.
  • Jika \( a < 0 \), grafik terbuka ke bawah.
  • Semakin besar nilai \( |a| \), grafik semakin sempit.

Koefisien b:

Koefisien b berpengaruh pada posisi sumbu simetri dan titik puncak fungsi. Misalnya, pada persamaan \( y = x^2 + 2x + 4 \), bentuk kuadrat sempurnanya adalah \( (x + 1)^2 + 5 \).

Jika c positif: sumbu simetri \( x = -c \), titik puncak (-c, d).

Jika c negatif: sumbu simetri \( x = c \), titik puncak (c, d).

Koefisien c:

Konstanta c mempengaruhi posisi grafik terhadap sumbu Y:

  • Jika c semakin besar, grafik semakin berada di atas.
  • Jika c semakin kecil, grafik semakin berada di bawah.

Hubungan Diskriminan dengan Grafik

Diskriminan \( D \) digunakan untuk menentukan jumlah solusi persamaan kuadrat dan berhubungan dengan jumlah titik potong grafik terhadap sumbu X.

  • Jika D > 0: Grafik menyentuh sumbu X di dua titik (dua solusi real).
  • Jika D = 0: Grafik menyentuh sumbu X di satu titik (satu solusi real).
  • Jika D < 0: Grafik tidak menyentuh sumbu X (tidak ada solusi real).

Contoh: Pada persamaan \( y = 3x^2 + x + 1 \), diskriminannya adalah \( D = b^2 - 4ac = 1^2 - 4(3)(1) = -11 \), sehingga grafiknya tidak menyentuh sumbu X.
